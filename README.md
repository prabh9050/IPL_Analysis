# 🏏 IPL Analysis Dashboard 2008-2024

End-to-end data analysis of IPL matches using Python, SQL, and Power BI.

## 📊 Key Insights
- Mumbai Indians are the most successful IPL franchise (144 wins)
- Toss winning teams win only 36% of matches — toss impact is minimal
- Teams increasingly prefer to field first, especially post-2017
- David Warner has the highest batting average among top run scorers
- Wankhede Stadium hosts the most IPL matches

## 🛠️ Tech Stack
Python | Pandas | NumPy | Matplotlib | Seaborn | SQL | SQLite | Power BI

## 📁 Project Structure
- `data/` - IPL datasets (matches + deliveries)
- `notebooks/` - EDA and SQL analysis notebooks
- `visuals/` - Charts and visualizations
- `dashboard/` - Power BI dashboard (.pbix) and PDF export
